Daily Concept: Gaze patterns in human-AI interaction

1. What is it?
Gaze patterns in human-AI interaction refer to the temporal and spatial characteristics of eye movements exhibited by users while engaging with AI systems. These patterns serve as implicit behavioral signals that reveal underlying cognitive processes like attention allocation, cognitive load, and trust formation during tasks involving LLM reading or prompt formulation.

2. How does it work?
The mechanism relies on capturing high-frequency eye movement data to map raw coordinates onto semantic regions of interest (ROIs) on a screen. Researchers analyze metrics like fixation duration to quantify processing depth, where longer durations often correlate with higher cognitive load or difficulty in interpreting AI outputs. For example, a scanpath \( S \) can be modeled as a sequence of fixation coordinates \( S = (f_1, f_2, ..., f_n) \) where each \( f_i \) represents a spatial pause over a specific time window. Machine learning models then classify these sequences to detect user intent or confusion states without requiring explicit user feedback.

3. Why does it matter?
This concept matters because it provides a non-invasive channel for implicit feedback, allowing AI systems to adapt their responses based on real-time user comprehension. In the context of LLMs, gaze data can identify hallucinations or unclear explanations by detecting prolonged fixations or regressive saccades. This enables the development of attentive user interfaces that dynamically adjust content density or explanation style to align with the user's current cognitive state.

4. Key insight
Gaze patterns transform human-AI interaction from a unidirectional command-response loop into a bidirectional communication channel where the AI perceives user understanding in real time.

Paper Summaries

SIAgent: Spatial Interaction Agent via LLM-powered Eye-Hand Motion Intent Understanding in VR

Authors: Zhimin Wang, Chenyu Gu, Feng Lu
arXiv: 2603.00522v1

Summary

1. Research Question
Current VR interaction paradigms require users to memorize specific gestures to trigger actions, increasing cognitive load and error rates. This research investigates whether a system powered by Large Language Models (LLMs) can accurately infer user intent directly from natural eye-hand coordination without predefined gestures.

2. Positioning
The work challenges the dominant "Operation-to-Intent" paradigm in VR, where users must map specific inputs to specific commands. It fills a critical gap by leveraging recent advances in LLMs to interpret natural user behavior, moving away from rigid command structures toward flexible intent understanding.

3. Key Contribution
The primary contribution is the SIAgent framework, which introduces an "Intent-to-Operation" workflow that translates raw spatial interaction data into natural language for an LLM to interpret. This approach eliminates the need for gesture memorization and demonstrates superior intent recognition accuracy (97.2%) compared to traditional gaze-pinch methods (93.1%) while reducing physical fatigue.

4. Methodology
The proposed system consists of an intent recognition module that converts eye-tracking and hand motion data into natural language descriptions, which are processed by an LLM to deduce user goals. Once the intent is established, an agent-based execution module generates and performs the corresponding virtual tasks, bridging the gap between raw spatial input and system commands.

5. Limitations
While the abstract highlights high accuracy, potential limitations inherent to LLM-based systems include latency issues that could affect real-time interaction fluidity in VR. Additionally, the scope of the user study, covering over 60 tasks, may not fully capture the complexity of highly diverse or ambiguous real-world scenarios where intent recognition could fail.

6. Critical Evaluation
This work represents a significant step forward in applying LLMs to human-computer interaction, effectively using multimodal sensors (eye and hand tracking) to create a more intuitive user experience. The strength lies in shifting the burden of "correctness" from the user to the AI, though the practical deployment will depend heavily on the inference speed and cost of the underlying language models.

IRIS: Intent Resolution via Inference-time Saccades for Open-Ended VQA in Large Vision-Language Models

Authors: Parsa Madinei, Srijita Karmakar, Russell Cohen Hoffing et al.
arXiv: 2602.16138v1

Summary

1. Research Question
How can ambiguity in open-ended Visual Question Answering (VQA) be resolved without additional model training? The paper specifically explores whether real-time eye-tracking data can serve as a signal to clarify user intent for Large Vision-Language Models (VLMs).

2. Positioning
The paper positions itself at the intersection of cognitive science and multimodal AI, addressing the limitation of current VLMs that struggle with underspecified queries. It fills a critical gap by introducing a mechanism to incorporate implicit human intent, unlike prior work that relies solely on explicit text or image inputs.

3. Key Contribution
The main contribution is IRIS, a novel, training-free inference technique that leverages gaze data to boost VLM performance significantly. The authors also provide a new benchmark dataset and a real-time interactive protocol, validating that fixations aligned with speech onset are the most disambiguating.

4. Methodology
The proposed method utilizes real-time eye-tracking to identify specific image regions a user fixates on while asking a question. It uses these "saccades" and fixation points to guide the VLM's attention, effectively cropping or weighting the visual input to match the user's focus.

5. Limitations
While not explicitly detailed in the abstract, the approach inherently requires eye-tracking hardware, limiting accessibility on standard devices without such sensors. Additionally, the effectiveness relies on the precise synchronization between the start of the verbal question and the capture of fixation data.

6. Critical Evaluation
This work offers a compelling demonstration of how implicit physiological signals can augment LLM reasoning, evidenced by the substantial accuracy jump on ambiguous queries. The training-free aspect makes it highly adaptable, though the hardware requirement presents a barrier to widespread consumer application compared to software-only solutions.

Human-Centered Multimodal Fusion for Sexism Detection in Memes with Eye-Tracking, Heart Rate, and EEG Signals

Authors: Iván Arcos, Paolo Rosso, Elena Gomis-Vicent
arXiv: 2602.23862v1

Summary

### 1. Research Question. The paper investigates whether integrating human physiological signals with standard visual and textual analysis can improve the automated detection of sexism in memes. It specifically asks if implicit human reactions, such as cognitive load and attention patterns, provide an objective signal that content-only models miss.

### 2. Positioning. This work positions itself against standard content-only models that struggle with the ambiguity, cultural nuance, and humor often used to mask sexism. It fills a critical gap by moving beyond pixel and text analysis to a human-centered paradigm. The authors argue that human perception provides a ground truth that pure computational features cannot capture.

### 3. Key Contribution. The primary contribution is a novel dataset combining Eye-Tracking, Heart Rate, and EEG signals recorded from subjects viewing memes. The authors also propose a multimodal fusion architecture that integrates these physiological inputs with Vision-Language Model (VLM) features. This approach yields a statistically significant improvement in detection accuracy, particularly for nuanced categories of sexism.

### 4. Methodology. The researchers collected data from 16 subjects viewing 3984 memes, analyzing physiological differences between responses to sexist and non-sexist content. They found that sexist memes triggered higher cognitive load, evidenced by increased fixation counts and specific EEG spectral power changes. These signals were fused with enriched text-visual features from a VLM to train a binary classification model.

### 5. Limitations. The study is limited by a small sample size of only 16 participants, which may not fully represent the diversity of human perception. Additionally, the reliance on specialized hardware like EEG headsets and eye-trackers makes the method difficult to scale for real-time content moderation on consumer devices.

### 6. Critical Evaluation. This paper offers a compelling proof-of-concept that gaze and neural signals act as reliable indicators of subconscious perception, effectively bridging the gap between AI analysis and human understanding. While the performance gains are impressive, the practical application is currently restricted by the invasive nature of the data collection hardware.

GazeXPErT: An Expert Eye-tracking Dataset for Interpretable and Explainable AI in Oncologic FDG-PET/CT Scans

Authors: Joy T Wu, Daniel Beckmann, Sarah Miller et al.
arXiv: 2603.00162v1

Summary

1. Research Question
This paper addresses the critical shortage of expert radiologists and the lack of interpretability in current AI diagnostic tools for FDG-PET/CT scans. It investigates how capturing and integrating expert eye-tracking data can improve the performance and explainability of AI models in tumor detection tasks.

2. Positioning
While existing AI research focuses heavily on standalone lesion segmentation, these models often fail to integrate into clinical workflows due to reliability concerns and "black box" opacity. This work positions itself at the intersection of human-computer interaction and medical imaging, filling the gap between raw algorithmic performance and the cognitive processes of human experts.

3. Key Contribution
The primary contribution is the release of GazeXPErT, a comprehensive 4D eye-tracking dataset featuring 9,030 gaze-to-lesion trajectories derived from 346 FDG-PET/CT scans. The authors demonstrate the dataset's value by showing that incorporating gaze patterns improves 3D tumor segmentation DICE scores from \(0.6008\) to \(0.6819\). They also provide baseline results showing that Vision Transformers can effectively leverage this data for dynamic lesion localization.

4. Methodology
The researchers collected 60Hz eye-tracking data from both trainees and board-certified specialists using a custom annotation platform that simulates routine clinical reads. This raw data was synchronized with PET/CT image slices and converted into COCO-style formats suitable for machine learning. The team validated the dataset by training a 3D nnUNet model augmented with gaze data and Vision Transformers for intention prediction tasks.

5. Limitations
The dataset is specifically tailored to FDG-PET/CT imaging, meaning the generalizability of the gaze patterns to other imaging modalities like MRI or CT alone remains unverified. While the dataset is rich, the baseline experiments focus primarily on segmentation and localization, leaving other potential applications like causal reasoning or advanced visual grounding largely unexplored in this initial study.

6. Critical Evaluation
This work offers a significant resource for the medical AI community by effectively quantifying the "search process" of experts, which is often lost in standard bounding-box annotations. The reported performance boost in segmentation is substantial, suggesting that gaze data provides a strong prior for model attention. However, the reliance on specialized eye-tracking hardware may pose a barrier to widespread data collection replication in diverse clinical settings.

EyeLayer: Integrating Human Attention Patterns into LLM-Based Code Summarization

Authors: Jiahao Zhang, Yifan Zhang, Kevin Leach et al.
arXiv: 2602.22368v1

Summary

### 1. Research Question. The paper investigates whether human eye-gaze patterns can serve as a proxy for expertise to enhance Large Language Model (LLM) performance in code summarization. It specifically asks if integrating human attention mechanisms into LLMs can improve their ability to generate accurate natural language descriptions of source code.

### 2. Positioning. While LLMs have advanced significantly in code understanding, they often lack the specific focus patterns humans naturally employ when reading and comprehending logic. This work positions itself at the intersection of cognitive science and natural language processing, filling a gap by using eye-tracking data to guide the model's "attention" rather than relying solely on data-driven self-attention mechanisms.

### 3. Key Contribution. The primary contribution is EyeLayer, a lightweight module that seamlessly integrates human gaze data into various LLM architectures without disrupting their pre-trained representations. The novelty lies in modeling gaze as a Multimodal Gaussian Mixture to create attention priors, resulting in consistent performance gains across diverse model families like LLaMA-3.2 and CodeBERT.

### 4. Methodology. EyeLayer models human attention using a Multimodal Gaussian Mixture, utilizing learned parameters \((\mu_i, \sigma_i^2)\) to represent the location and intensity of developer focus. These parameters guide the redistribution of token embeddings, effectively teaching the model to prioritize code tokens that humans naturally focus on during reading tasks.

### 5. Limitations. A primary limitation is the dependency on high-quality eye-tracking datasets, which are notoriously difficult and expensive to collect compared to standard code-text pairs. Additionally, the effectiveness of the learned gaze priors may vary when applied to programming languages or coding styles that differ significantly from the training data.

### 6. Critical Evaluation. This work offers a compelling proof-of-concept for augmenting machine attention with biological signals, moving beyond standard fine-tuning approaches. The architecture-agnostic design is a significant strength, suggesting that human gaze encodes structural information about code that pure statistical models often miss.
